# airy-canvas-mini

> [airy-canvas](https://github.com/Airyworks/airy-canvas-mini) mini demo

## Quick start

```
docker run \
  -p 3333:3000 \
  -d \
  you06/airy-canvas-mini
```

## Manual build

```sh
git clone https://github.com/Airyworks/airy-canvas-mini.git
cd airy-canvas-mini
yarn && yarn build
yarn start
```
